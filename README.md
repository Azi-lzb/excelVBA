# 通用汇总工具 VBA Excel

基于 Excel VBA 的报表处理工具箱，面向批量报表加工场景（机构标准化、时序提取、打印输出、去重追加、VBA 注入等）。

## 主要能力

- `2.x 一二批功能`：机构名称标准化、外资行剔除、外汇页眉处理、工作表提取、地区核对等。
- `3.9`：时序提取、维度比对、宽表汇总。
- `3.10`：按批注/按配置打印输出（保留源格式、轻度重排、快速复制）。
- `3.11`：按批注/按配置查重、删重、去重追加。
- `5.x`：VBA 同步、注入与清理。
- `6.x`：初始化配置表。

## 仓库结构

- `VBA_Export/Modules`：核心 `.bas` 模块源码。
- `VBA_Export/Documents`：`ThisWorkbook.cls` 与工作表代码导出。
- `VBA_Export/InitData`：初始化配置模板（`*.tsv`）。
- `scripts`：辅助脚本（编码转换、批量处理等）。
- `VbaV2.3.4.xlsm`：工具箱主工作簿（可直接运行）。
- `使用说明.md`：详细功能说明与操作手册。

## 快速开始

1. 打开 `VbaV2.3.4.xlsm` 并启用宏。  
2. 在 Excel 信任中心启用：`信任对 VBA 工程对象模型的访问`。  
3. 按需先执行 `6.x` 初始化配置表。  
4. 通过菜单 `报表工具(&S)` 运行对应功能。

## 开发与编码说明

- VBA 导出文件以 `GBK/ANSI + CRLF` 为主，避免中文乱码与导入异常。
- 修改 `.bas/.cls` 后，建议执行：

```bash
python convert.py
```

用于统一编码/换行格式，保证可被 VBE 正常导入。
