# 通用汇总工具（VBA + Python）

本项目用于金融报表场景的批量处理，核心目标是：
- 用配置表驱动批处理，减少手工操作。
- 统一“时序提取 / 宽表汇总 / 打印输出 / 去重追加”口径。
- 同时提供 VBA（Excel 内）与 Python（外部批跑）两套实现。

---

## 1. 项目结构

- `VbaV2.4.2.xlsm`：主工具箱（菜单入口）。
- `VBA_Export/Modules/`：VBA 模块源码（`.bas`）。
- `VBA_Export/Documents/ThisWorkbook.cls`：菜单与入口绑定。
- `VBA_Export/InitData/`：初始化配置模板（`.tsv`）。
- `pytools/`：Python 版工具（终端菜单、批处理、COM/非COM双引擎）。
- `使用说明.md`：详细功能说明（逐菜单口径）。

---

## 2. 主要功能

### 2.1 时序工具（3.9）
- 表头路径比对（按位置 + 按路径）。
- 时序提取（快速精简）。
- 宽表规则汇总（行头加后缀 / 不加后缀）。
- 路径标准化映射（行头/列头统一口径）。

### 2.2 打印工具（3.10）
- 按批注打印（保留格式 / 轻度重排 / 快速复制）。
- 按配置打印（按模式执行，支持写入目标工作簿）。
- 方向控制：默认宽高比判定，可批注/配置覆盖。

### 2.3 去重工具（3.11）
- 按批注检查重复（标红）。
- 按批注删除重复行。
- 按配置检查 / 删除重复。
- 按配置去重追加到目标（支持目标不存在时自动新建）。

### 2.4 其他常用工具
- 按批注汇总 / UsedRange 汇总。
- 批量重命名文件、批量改 Sheet 名。
- 初始化配置与批量导入导出模块。

## 3. 快速上手（VBA）

1. 打开 `VbaV2.4.2.xlsm`，启用宏。  
2. 先执行初始化菜单（将 `InitData` 模板写入配置表）。  
3. 按功能填写配置表（时序提取规则、路径标准化映射、打印配置、去重追加配置等）。  
4. 回到菜单执行对应功能。  
5. 结果默认输出到配置指定路径；未指定时按功能回退到临时结果簿或默认输出目录。  

---

## 4. Python 版（pytools）

`pytools` 提供同类批处理能力，适合大批量/低交互场景：
- 支持非 COM 与 COM 引擎。
- 支持自动路由（如 `.xls` 走 COM，其它走非 COM）。
- 终端菜单执行，配置文件统一走 `config/config.xlsx`。

运行示例（源码方式）：

```powershell
cd pytools
python main.py
```

---

## 5. 配置与口径建议

- 同类报表优先固化为规则配置，避免临时手工筛选。
- 映射表只做“标准化”，不直接承担业务去重逻辑。
- 宽表汇总建议使用“全列去重”口径，降低误删风险。
- 打印配置建议固定模板 + 月度替换源文件路径。

---

## 6. 注意事项

- VBA 模块文件（`.bas/.cls/.frm`）请保持 ANSI/GBK 编码导入导出。
- 修改 `ThisWorkbook` 菜单绑定后，需同步检查宏入口名是否一致。
- 如果 Git 提交中包含敏感文件，请在提交前先通过 `.gitignore` 或手动排除。
